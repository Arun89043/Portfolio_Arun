Project 3

Developed a transformer-based image captioning system that generates meaningful and context-aware captions for images using Vision Transformer and GPT-2 models.

Live Link

Project Overview

The Transformer-Based Image Captioning System is designed to automatically generate accurate and descriptive captions for images using state-of-the-art deep learning techniques. The system was fine-tuned on the Flickr8k dataset to learn visual and linguistic relationships between images and natural language.

A Vision Transformer (ViT) was used to encode image patches into meaningful embeddings, while GPT-2 was employed for autoregressive caption generation. Beam search decoding was applied to improve the quality and coherence of generated captions.

A complete inference pipeline was built and deployed using a Gradio-based interface, enabling real-time image uploads and caption display. Mixed-precision training and optimized preprocessing techniques were applied to reduce memory usage and improve inference speed, making the system suitable for consumer-grade hardware.
