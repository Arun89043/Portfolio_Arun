Project Overview

The Gesture to Speech Communication System is an end-to-end real-time application designed to assist speech-impaired individuals by translating hand gestures into spoken language. The system uses a standard webcam to capture gestures and applies computer vision techniques to interpret hand movements accurately.

A custom Convolutional Neural Network (CNN) was trained on a self-curated dataset containing 16 distinct gesture classes. MediaPipe was integrated to extract 21-point hand landmarks, which were normalized to improve robustness and consistency across different environments.

The recognized gestures are mapped to corresponding text and converted into natural speech using Google Text-to-Speech (gTTS). The application is deployed using Flask, providing a simple and interactive web interface. Optimized for low latency and cross-platform usage, the system eliminates the need for specialized hardware and runs efficiently on consumer devices.
